# Ferns-and-Petals-Sales-Analysis-
Developed an Excel dashboard to analyze Ferns and Petals sales data, uncovering trends in revenue, customer behavior, and product performance.
**📌 Project Overview**


This project focuses on analyzing Ferns and Petals (FNP) sales data using Microsoft Excel to uncover valuable business insights. The analysis helps understand customer purchasing behavior, sales trends, product performance, and revenue generation through an interactive Excel dashboard.

**🎯 Project Objectives**

📈 Analyze overall sales performance.
💰 Track revenue and order trends.
🎁 Identify top-selling products and categories.
👥 Understand customer purchasing patterns.
📅 Analyze seasonal and monthly sales trends.
📊 Create an interactive dashboard for business insights.

**🛠️ Tools & Techniques Used** 

📗 Microsoft Excel
📊 Pivot Tables
📈 Pivot Charts
🔍 Data Cleaning
🧮 Excel Functions & Formulas
🎨 Conditional Formatting
📋 Dashboard Design
